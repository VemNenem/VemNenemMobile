# VemNenem

Este é um projeto Expo criado com [`create-expo-app`](https://docs.expo.dev/more/create-expo/).

## Começando

1. Instale as dependências

```bash
npm install
```
2. Inicie o app
   
```bash
npx expo start
```

3. Recursos úteis

```
Documentação do Expo
Guias do Expo
Tutorial oficial do Expo
```
