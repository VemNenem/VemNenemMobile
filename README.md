# VemNenem

Este é um projeto Expo criado com [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Começando

1. Instale as dependências

```bash
npm install
```
2. Inicie o app
   
```bash
npx expo start
```

3. Recursos úteis

```bash
Documentação do Expo
Guias do Expo
Tutorial oficial do Expo
```
